## Vim
### 三种模式
  1. 命令模式：控制光标移动，可对文本进行复制、粘贴、删除和查找等工作。
  2. 输入模式：正常的文本录入。
  3. 末行模式：保存或退出文档，以及设置编辑环境。
### 基本命令
  1. dd 相当于windos crtl + X 剪切 5dd 删除5行
  2. yy 复制 5yy 复制5行
  3. u 撤销
  4. p 粘贴
### 末行模式常用命令
  1. :w 保存
  2. :q 退出
  3. :q！ 强制不保存退出
  4. :wq! 强制保存退出
  5. :set nu 显示行号
  6. :set nonu 不显示行号
  7. :set 323 跳转
  8. :s/one/two	将当前光标所在行的第一个one替换成two
  9. :s/one/two/g 替换所有当前行
  10. :%s/one/two/g 替换所有
  11. /搜索 ?
